# Cryptoto

## Prerequisites

Install `pipx`:
```bash
python -m pip install --user pipx
```

Install `Poetry`:
```bash
pipx install poetry
pipx ensurepath
```

## Installation

Install dependancies:
```bash
poetry install --no-root
```

## Development

For your IDE:
- Remember to select the right interpreter (the poetry one)

Installing packages (example):
```bash
poetry add example-package-here
```
